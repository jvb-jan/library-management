# **App Name**: BiblioFlow

## Core Features:

- Intelligent Access Shield: Enterprise-grade role-based access control (RBAC) leveraging JWT tokens to secure Admin, Librarian, and User environments.
- AI Book Curator: A tool that uses generative AI to analyze book titles and metadata to produce professional, engaging book descriptions and summaries automatically.
- Live Inventory Nexus: Comprehensive CRUD system for books stored in a MySQL database, allowing for real-time tracking of AVAILABLE, BORROWED, and OUT_OF_STOCK statuses.
- Visual Analytics Dashboard: High-fidelity statistics cards visualizing total counts of users, roles, and book statuses for administrative oversight.
- Universal Omni-Search: Dynamic filtering and search capabilities enabling users to drill down by genre, author, title, and current availability status.
- Responsive Glassmorphism UI: A sleek, responsive layout featuring dark mode support, glassmorphic card elements, and specialized views for different user roles.
- System Feedback Engine: Integrated global exception handling system that delivers success and error feedback via modern toast notifications and custom loading indicators.

## Style Guidelines:

- Primary color: Electric Amethyst (#BA75FF) to evoke a sense of digital wisdom and modern architecture.
- Background: Midnight Graphite (#1F1825), a deeply desaturated dark theme that emphasizes the high-contrast accent elements.
- Accent color: Azure Velocity (#637EFF), an analogous hue used for interactive states and primary call-to-action buttons.
- A professional font pairing of 'Space Grotesk' for headlines to maintain a tech-forward look, and 'Inter' for body text to ensure maximum legibility across denser data tables.
- Feather-light outline icons with consistent stroke weights to complement the minimalistic glassmorphism design.
- Modern sidebar-focused architecture with sticky top navigation and responsive grid containers for dashboard analytics.
- Fluid spring animations for modal transitions and subtle stagger effects when loading inventory lists to create an organic user experience.
