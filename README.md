# Library Management System

This project is a role-based Library Management System.

## Authentication & Authorization

The system includes authentication and authorization with three different roles:

1. Admin
2. Librarian
3. User

## Roles & Permissions

### Admin
- Full access to the system
- Can Create, Read, Update, and Delete (CRUD)
- Manage books and users

### Librarian
- Can only View and Update records
- Cannot create or delete records

### User
- Can only View available books and records

## Features

- Secure Login System
- Role-Based Access Control
- Separate Dashboards for:
  - Admin
  - Librarian
  - User
- Protected Pages
- Book Management System

## Pages Included

- Login Page
- Admin Dashboard
- Librarian Dashboard
